# 超初級日本語学習者向け リスニング・発音練習法

## 概要

日本語がほとんどできない留学生を対象とした、スマートフォンで隙間時間にできるリスニング・発音練習の学習法。

### 対象者

- 日本語超初級者（N5以下）
- リスニングがままならず、授業についていけないレベル
- 日本に来たばかりの留学生

### 設計方針

| 方針 | 理由 |
|------|------|
| 採点しない | できないことが数値化されるとモチベーションが下がる |
| アドバイス型 | 「伝わった」を前提に、改善点を具体的に伝える |
| 短時間で完結 | 1回15〜20分、隙間時間に取り組める |
| 母語サポート | 超初級者には母語での説明が効果的 |

---

## 5ステップ学習法

### 全体フロー

```
Step 1: 音声だけ聞く（現在地の確認）
    ↓
Step 2: スクリプト・翻訳で意味を理解
    ↓
Step 3: 意味がわかった状態でもう一度聞く
    ↓
Step 4: 自分で声に出す（発音練習）
    ↓
Step 5: 録音してAIからアドバイスをもらう
```

---

## 各ステップの詳細

### Step 1：音声だけ聞く

#### 目的
今の自分がどこまで聞き取れるか確認する

#### やり方
- 短い音声（15〜30秒程度）を1回だけ通して聞く
- 聞き取れた単語、なんとなくわかった部分をメモ
- 「全然わからなかった」でもOK

#### 所要時間
約1〜2分

---

### Step 2：スクリプト・翻訳で意味を理解

#### 目的
「何を言っていたか」を完全に理解する

#### やり方
1. 日本語スクリプトを見る
2. 知らない単語を母語で確認
3. 文全体の意味を母語で理解
4. スクリプトを見ながら音声を聞く（音と文字を一致させる）

#### この段階でやること
- 知らなかった単語をリストアップ
- 「ここがつながって聞こえてたのか」と気づく
- 発音やイントネーションを意識して確認

#### 所要時間
約3〜5分

---

### Step 3：意味がわかった状態でもう一度聞く

#### 目的
意味がわかった状態で聞き、音と意味を脳内で結びつける

#### やり方
- スクリプトを見ないで聞く
- 2〜3回繰り返す
- 意味がわかりながら聞けることを確認

#### 所要時間
約2〜3分

---

### Step 4：自分で声に出す

#### 目的
発音・リズム・イントネーションを身につける

#### 3段階で進める

##### 4-a：オーバーラッピング（初心者はここから）
音声と**同時に**スクリプトを見ながら声に出す

- 音声にぴったり合わせることで、スピード・リズム・イントネーションが身につく
- スクリプトを見ていいので、初心者でもできる
- 3回程度繰り返す

##### 4-b：リピーティング（少し慣れたら）
音声を**一文ずつ止めて**、真似して言う

- 自分のペースで丁寧に発音できる
- 長い文は区切ってOK
- 3回程度繰り返す

##### 4-c：シャドーイング（上級者向け・任意）
音声を聞きながら、0.5〜1秒遅れで追いかける

- 超初級者はここまで行かなくてOK
- 4-a、4-bだけでも十分効果あり

#### 所要時間
約4〜6分

---

### Step 5：録音してAIからアドバイスをもらう

#### 目的
- 自分の発音を客観的に確認する
- 「伝わった」という自信を持つ
- 次に意識するポイントを1つ知る

#### フロー（方法2＋方法3のハイブリッド）

```
① 自分の発音を録音する
    ↓
② AIの認識結果を見る（客観的フィードバック）
    ↓
③ 自己チェック（選択式で振り返り）
    ↓
④ AIからアドバイスを受け取る
    ↓
⑤ （任意）お手本と自分の録音を聞き比べる
```

#### 所要時間
約3〜5分

---

## Step 5 の詳細仕様

### 5-1：録音

#### UI要件
- 大きな録音ボタン
- 録音中は視覚的フィードバック（波形表示など）
- 録音完了後、再生して確認できる
- やり直しボタン

---

### 5-2：AIの認識結果表示

#### 目的
学生が自分の発音を客観的に見られるようにする

#### 表示内容
```
お手本：きのう、ともだちと えいがを みました

あなたの発音（AIが聞き取った結果）：
「きのう、ともだちと えーがお みました」
```

#### 技術要件
- 音声認識API（Whisper等）でテキスト化
- お手本テキストと認識結果を並べて表示
- 差分がある部分をハイライト（任意）

---

### 5-3：自己チェック（選択式）

#### 目的
- 学生自身に振り返りの習慣をつける
- AIアドバイスの参考情報にする

#### UI
```
自分の発音、どうでしたか？

○ お手本と同じように言えた
○ だいたい言えたけど、少し違った
○ 難しかった
○ わからない
```

#### 選択肢の意図
| 選択肢 | 学生の状態 | AIの対応 |
|--------|-----------|---------|
| 同じように言えた | 自信あり | 良かった点を強調、さらなる改善点を1つ |
| だいたい言えた | 課題を認識 | 具体的な改善ポイントを提示 |
| 難しかった | 困っている | 励まし＋基本的なアドバイス |
| わからない | 判断できない | 聞き比べを促す＋基本アドバイス |

---

### 5-4：AIアドバイス

#### フィードバックの構成

```
① まず「伝わったこと」を認める
    ↓
② 良かった点を具体的に1〜2個伝える
    ↓
③ 「もっと良くなるヒント」を1個だけ
    ↓
④ 励ましで終わる
```

#### アドバイス例

```
━━━━━━━━━━━━━━━━━━━━
「きのう、ともだちと えいがを みました」
ちゃんと伝わりましたよ！

👍 良かったところ
・「ともだち」の発音、とてもきれいです
・文のリズムが自然でした

💡 もっと良くなるヒント
・「えいが」の「え」を、もう少し口を横に開いて
　発音してみてください

この調子で続けていきましょう！
━━━━━━━━━━━━━━━━━━━━
```

#### AIプロンプト設計方針

```
あなたは日本語学習をサポートする優しい先生です。
学生の発音をお手本と比較して、アドバイスしてください。

ルール：
- 最初に「伝わった」ことを認める
- 良かった点を1〜2個見つける
- 改善点は1個だけ、具体的に伝える
- 点数やスコアは絶対につけない
- 励ましの言葉で終わる
- やさしい日本語で書く（N4-N5レベル）
- 必要に応じて母語での補足を入れる

入力情報：
- お手本テキスト
- 学生の発音（音声認識結果）
- 学生の自己評価
- 学生の母語（任意）
```

#### 母語別アドバイス（任意機能）

特定の母語話者に共通する発音の癖に対応したアドバイスを提供

```
🇻🇳 ベトナム語話者向け
「えいが」の「え」は、ベトナム語の「ê」に近いです。
「e」ではなく「ê」のつもりで発音してみてください。

🇨🇳 中国語話者向け
「映画」は中国語と同じ漢字ですが、
日本語では「ei-ga」と2拍で発音します。
```

---

### 5-5：聞き比べ機能（任意）

#### 目的
もっと練習したい学生が、自分の耳で違いに気づけるようにする

#### UI
```
┌─────────────────────────┐
│ ▶ お手本を聞く          │
│ ▶ 自分の録音を聞く       │
│ ▶ 両方を交互に聞く       │
└─────────────────────────┘
```

#### 表示するポイント
```
💡 この文のポイント
・「えいが」→「え」は口を横に広げる
・「を」→「お」と同じ音でOK
・ゆっくりでも大丈夫！
```

## 1回の練習タイムライン

| 時間 | Step | 内容 |
|------|------|------|
| 0:00 | 1 | 音声を1回聞く（字幕なし） |
| 2:00 | 2 | スクリプト・翻訳を確認、単語を調べる |
| 7:00 | 3 | 意味がわかった状態で2〜3回聞く |
| 10:00 | 4 | オーバーラッピング・リピーティング |
| 16:00 | 5 | 録音 → AI認識結果 → 自己チェック → アドバイス |
| 20:00 | - | （任意）聞き比べ、再練習 |

**合計：15〜20分**

---

## 音声素材の要件

### 長さ
- 15〜30秒程度
- 1〜3文程度

### レベル
- N5〜N4レベルの語彙・文法
- ゆっくり、はっきりした発音

### 必要なデータ
| 項目 | 内容 |
|------|------|
| 音声ファイル | MP3/WAV形式 |
| スクリプト | ひらがな表記（漢字にはふりがな） |
| 翻訳 | 対応言語（英語、ベトナム語、中国語等） |
| 発音ポイント | この文で注意すべき音のリスト |

---

## 技術要件（概要）

### フロントエンド
- スマートフォン対応（PWA or ネイティブアプリ）
- 音声録音機能
- 音声再生機能（速度調整あり）

### バックエンド
- 音声認識API（Whisper推奨）
- LLM API（Claude/GPT）でアドバイス生成
- 学習履歴の保存

### データ
- 音声素材データベース
- ユーザーの学習履歴
- 録音データ（一時保存 or 保存しない）
